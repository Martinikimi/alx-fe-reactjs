# Recipe Sharing Platform

A modern recipe sharing platform built with React and Tailwind CSS.

## Features

- Browse recipes
- View detailed recipe information
- Submit new recipes
- Responsive design

## Tech Stack

- React 18
- Tailwind CSS
- Vite

## Getting Started

### Prerequisites

- Node.js (version 16 or higher)
- npm or yarn
