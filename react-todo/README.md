# React Todo List with Testing - ALX Project

A fully tested Todo List application built with React, Jest, and React Testing Library.

## Features

- ✅ Add new todos
- ✅ Toggle todo completion status (click on todo or checkbox)
- ✅ Delete todos
- ✅ Statistics display (total, completed, pending)
- ✅ Empty state handling
- ✅ Form validation (no empty todos)
- ✅ Fully tested with Jest & React Testing Library

## Technologies Used

- React 18
- Jest 29
- React Testing Library
- Vite (build tool)
- CSS3
